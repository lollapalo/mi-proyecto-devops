Acerca del Proyecto Final

La finalidad del proyecto es:

  • Aplicar un flujo de trabajo colaborativo con GitHub
  • Utilizar ramas para desarrollar funcionalidades
  • Integrar cambios de diferentes integrantes
  • Simular un flujo DevOps básico

Objetivo del Proyecto

Este proyecto tiene como objetivo practicar el uso de Git y GitHub dentro de un flujo básico de DevOps.
